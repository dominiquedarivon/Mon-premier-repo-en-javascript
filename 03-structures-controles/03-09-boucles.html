Les boucles

Présentation des boucles

Les boucles vont nous permettre d'excécuter plusieurs fois un bloc de code . C'est à dire , d'exécuter un code en boucle tant qu'une condition donnée n'est pas vérifiée. Utiliser une boucle nous permet de n'écrire le code que l'on veut exécuter une seule fois.

Il existe 6 types de boucles en JS:

  • La boucle While : "tant que..."
  • La boucle do ... while: "fait ... tant que"
  • La boucle for:" pour"
  • La boucle for... in:" pour ... dans "
  • La boucle for...of:" pour parmi"
  • La boucle for ...await ... of: "pour ... en attente de ... parmi"

Le fonctionnement générale des boucles est toujours le même , on pose une condition qui sera également liée à la valeur d'une variable et on exécute notre boucle tant que la condition est vérifiée.


Pour éviter de rester bloqué à l'infini dans une boucle, vous pouvez donc déjà noter qu'il faudra que la condition donnée soit fausse à un moment pour pouvoir en sortir. Pour que notre condition devienne fausse, on pourra par exemple décrémenter ou incrémenter la valeur de votre variableà chaque nouveau passage dans la boucle.

Les boucles seront essentiellement composées de trois choses:

  • Une valeur de départ qui va servir à initialiser la boucle et qui servira de compteur
  • un text ou une condition de sortie qui précisera le caractère de sortie de la boucle
  • Un " itérateur" qui va modifier la valeur de départ de la boucle à chaque passage de celle-ci. Pour cela , on devra 'incrémenter' la valeur de départ.

Les opérateurs d'incrémentation et de décrémentation

incrémenter
Ajouter 1 à une valeur
Décrémenter
Enlever 1 à une valeur
Les opérateurs d'incrémentation et de décrémentation
Exemple Résultat
++x Pré-incrémentation : incrémente la valeur contenue dans la variable x, puis retourne la valeur incrémentée
x++ Post-incrémentation : retourne la valeur contenue dans x avant incrémentation, puis incrémente la valeur de $x
--x Pré-décrémentation : décrémente la valeur contenue dans la variable x, puis retourne la valeur décrémentée
x-- Post-décrémentation : retourne la valeur contenue dans x avant décrémentation, puis décrémente la valeur de $x

La boucle while, tant que en français, va nous permettre de répeter une série d'instructions tant qu'une condition donnée est vraie, c'est à dire tant que la condition de sortie n'est pas vérifiée.

Boucle do ... while

Cette boucle est relativement semblable à while, la seule différece réside dans l'ordre dans lequel vont se faire les opérations ; on effectuera un premier passage dans une boucle do... while même si la condition de sortie n'est pas vérifiée . C'est à dire que le code s'exécutera au moins une fois.
