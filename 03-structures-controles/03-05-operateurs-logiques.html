05 Les opérateurs Logiques

Que sont les opérateurs logiques ?

Ce type d'opérateurs va nous permettre d'effectuer plusieurs Comparaisons dans nos tests ou d'inverser la valeur logique du résultat d'un test.

1- Présentation et utilisation des opérateurs logiques

Le Javascript supporte trois opérateurs logiques :
l'opérateur logique "ET" OU &&
l'opérateur logique "OU" ou ||
l'opérateur logique "NON" ou !

Opérateurs logiques
Nom des opérateurs Symbole des opérateurs Description
And (ET) && Renvoie true si toutes les comparaisons sont évaluées à true, ou false dans le cas inverse
Or (OU) || Renvoie true si au moins l'une des comparaisons est évaluée à true ; sinon c'est false.
No (NON) ! Renvoie false si une comparaison est évaluée à true et renvoie true dans le cas contraire
