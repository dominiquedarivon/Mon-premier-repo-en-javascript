Introduction aux structures de contrôle

Les conditions et les boucles

Il existe deux types de contrôles de structures de base aue l'on trouve dans tous kes langages de programmation et donc en JS.Ces deux types de structures sont

  • Les Conditions
  • Les boucles (structures de contrôle de boucles)

Les conditions vont nous permettre d'exécuter un certain nombre d'instructions, si et seulement si une certaine condition est vérifiée. Les boucles quant à elles, vont nous permettre d'exécuter un bloc de code en boucles tant qu'une condition donnée n'est pas vérifiée

Les conditions

  • La condition simple:
    if () {}
    SI (x>5) { affiche ça}
  • La condition un peu plus complexe :
    if () {} else {};
    SI (x>5) {affiche çà} SINON {affiche plutôt ça }
  • La condition multiple :
    if () {} else if () {} else {}
    SI (x>5) {affiche ça} SINON SI (x=5){affiche plûtot ça} SINON {affiche moi ça}
