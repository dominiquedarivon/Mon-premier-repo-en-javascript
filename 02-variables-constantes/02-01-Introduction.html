Cours Javascript

Introduction

1-Qu'est -ce qu'une variable ?

Une variable est un conteneur servant à stocker des informations de manière temporaire:ça peut être du texte (une chaine de caractières), des nombres, etc.La variable varie! Elle stocke différentes valeurs au fil du temps

Une variable et sa valeur sont deux choses différentes.

2-Les règles de déclaration

On dit que l'on déclare une variable(comme en PHP). On va déclarer une variable grâce à des mots clés.

A- Les mots clés

    Let : EXEMPLE
    let age = 28;
  • var : EXEMPLE
    var couleur = 'jaune';

B - Quelques règles

  • Le nom d'une variable doit toujours commencer par une lettre ou un underscore
  • Le nom d'une variable ne doit contenir de caractères spéciaux
  • il n'y a pas d'espace dans la déclaration d'une variable

3- initialiser une

Quand on assigne une variable à une varuable vide, on dit que l'on stocke une valeur ou qu'on initialise une variable

4- Différences entre "var" et "let"

let est plus récent que var par défaut, nous utiliserons let.

A- La remontée des variables (avec var)

var, nous ne sommes pas obligés de déclarer la variable avant de la manipuler ou de l'affecter :

  • Ceci fonctionne:
    nom = 'Perinel';
    var nom ;
  • Ceci ne fonctionne pas :
    ville = 'paris' ;
    let ville ;

B- la redéclaration des variables

Avec var, on avait la possibilité de déclarer une même variable en utilisant à chaque fois var, ce qui avait pour effet de modifier la valeur stockée. On ne peut pas le faire avec let.

  • Ceci fonctionne
    var chien = 'max';
    var chien ='Medor';
  • Ceci ne fonctionne pas
    let chat ='obi-wan';
    let chat =' Skywalker';

C- la portée des variables

La portée d'une variable désigne l'endroit ou cette variable va pouvoir être utilisée dans un script. Nous rentrerons plus dans les détails lorsque nous verrons les fonctions en Javascript

Conclusion

La syntaxe de déclaration des variables avec let est la nouvelle syntaxe. var est vouée à disparaitre. Vous devez donc normalement toujours utiliser le mot clef let

5- Utilisé des variables

Il sera pratique de stocker différents types de données et d'informations dans une variable.

Exemple: avec la fonction prompt(), on pourra récupérer une donnée envoyée par l'utilisateur et cette donnée sera stockée grâce à une variable.
