Les types de données

Il existe 7 types de données Javascript

String ou 'chaine de caractères'

En Js , on va utiliser indifféremment des guillemets doubles ou des apostrophes pour entourer une chaine de caractères.Ces méthodes seront strictements équivalentes à la différence du PHP par exemple

Faites attention cependant à un point , si votre chaine de caractères contient un caractère qui est le même que le délimiteur choisi, il faudra échaper ce caractère grâce à un antislash (\ -> alt Gr + 8), ou un autre délimiteur.

2- Number ou nombres

Les variables en JS vont également pouvoir stocker des nombres. ces nombres pourront être des nombres entiers positifs ou négatifs, ou à virgules (décimaux).

3- Boolean ou booléen

Ce type de donnée sera utile pour vérifier si un test est vrai ou faux. Un booléen, en algèbre , est une valeur binaire(soit 0 soit 1) . En informatique , le type booléen est un type qui ne contient que deux valeurs : TRUE ou FALSE

4- Null ou vide

Lorsqu'une variable est laissée vide , elle affiche le type null: cela signifie qu'elle n'a pas de contenu.

5- Undefined non definie

Si on déclare uen variable sans attribuer de valeur alors son type sera undelined

6/7- Symboles et objects

Les objets sont des structures complexes qui vont pouvoir stocker plusieurs valeurs en même temps et que nous étudierons plus tard car il s'agit là d'un sujet complexe à appréhender pour un débutant

Si on déclare une variable et qu'on lui passe la valeur null alors sont type sera objects
