Les constantes

Le Javascript supporte depuis plusieurs années l'utilisation des constantes.

1 - Définition et utilité des constantes

Une constante est similaire à une variable au sens ou c'est un conteneur pour une valeur , mais la difference est que la valeur ne peut pas être modifiée. C'est à dire une fois sa valeur définie elle sera inchangeable

2- Déclarer une constante en Javascript

Pour déclarer uen constante sur une page de Javascript, on utilisera lemot clef Const. Dans notre fichier JS, on déclare la constante exactement comme une variable , il est par contre obligatoire de l'initialiser lors de sa déclaration.

Par exemple on déclarera le code suivant :
const pays = "France.
Cela signifie que par la suite on ne pourra pas faire évoluer sa valeur comme c'est possible avec let et var.

Il sera interdit de faire ceci :
const pays = 'France';
pays= 'Angleterre';
